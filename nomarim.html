Part A — Definitions from Multiple Angles
D1
Positive Definite (PD) — 7 Equivalent Characterizations
A is symmetric throughout. All 7 are 100% equivalent.
▼
#1 Quadratic Form xᵀAx > 0 for all x ≠ 0 — the definition. Acts as a generalized "distance." DEF
#2 Eigenvalues All eigenvalues are strictly positive: λ₁, λ₂, …, λₙ > 0. How: use Spectral Theorem A = QΛQᵀ, then xᵀAx = zᵀΛz = Σλᵢzᵢ² > 0 iff all λᵢ > 0. EIGENVALUES
#3 LU Pivots All pivots in LU decomposition are > 0. (Without row swaps — if LU exists cleanly.) CH1
#4 Cholesky Success Cholesky decomposition A = LLᵀ succeeds with all diagonal entries of L strictly positive. If it fails midway → A was NOT PD. CH1
#5 Leading Minors All leading principal minors are > 0: det(A₁) > 0, det(A₂) > 0, …, det(A) > 0 where Aₖ is the top-left k×k block. EIGENVALUES
#6 Gram Matrix Form A = BᵀB for some matrix B with full column rank. Full rank is essential — without it you only get PSD. CH2
#7 Tall + Full Col Rank If A is tall with full column rank, then AᵀA is automatically PD. This is #6 with B = A. CH2
#8 Singular Values All singular values σᵢ > 0 AND A is symmetric. (For sym. A: σᵢ = |λᵢ|, and for symmetric PD specifically σᵢ = λᵢ.) SVD
PSD vs PD: Replace every ">" with "≥" to get Positive Semi-Definite (PSD). AᵀA is always PSD; it's PD only when A has full column rank.
D2
Invertible (Non-Singular) — 10 Equivalent Conditions
For square A: all of these live or die together. If one fails, all fail.
▼
#1 Definition There exists A⁻¹ such that AA⁻¹ = A⁻¹A = I DEF
#2 Determinant det(A) ≠ 0. Compute via: product of U diagonal (from LU) or product of eigenvalues. CH1
#3 Full Rank rank(A) = n (for n×n matrix) DEF
#4 Column Independence Columns of A are linearly independent DEF
#5 Trivial Null Space null(A) = {0} — the only solution to Ax = 0 is x = 0 DEF
#6 LU Pivots All pivots ≠ 0 in LU (no zero on diagonal of U) CH1
#7 Eigenvalues All eigenvalues ≠ 0 (since det(A) = ∏λᵢ) EIGENVALUES
#8 QR Diagonal All diagonal entries of R in QR decomposition are nonzero CH3
#9 Singular Values All singular values σᵢ > 0 — the most numerically stable check SVD
#10 Unique Solution Ax = b has exactly one solution for every b DEF
det(A) two ways: From LU → det(A) = (±1) · ∏ pivots_i (sign from P). From eigenvalues → det(A) = λ₁·λ₂·…·λₙ. Both give the same number.
D3
Unique LS Solution x̂ — The One Chain
All of these say the same thing in different tools' languages
▼
#1 Full Column Rank A has full column rank — the fundamental condition CORE
#2 AᵀA is PD AᵀA is PD → invertible → normal equations AᵀAx̂ = Aᵀb have unique solution CH2
#3 R diagonal nonzero Diagonal of R in QR has no zeros → x̂ = R⁻¹c₁ unique (back substitution) CH3
#4 Cholesky on AᵀA Cholesky succeeds on AᵀA → it's PD → unique x̂. Costs n³/6 after computing AᵀA. CH1
#5 All σᵢ > 0 SVD: all singular values positive → x̂ = VΣ⁻¹Uᵀb unique SVD
#6 null(A) = {0} Trivial null space → no free variables in normal equations → unique solution DEF
Full col rank ⟺ R diagonal nonzero ⟺ AᵀA is PD ⟺ Cholesky works on AᵀA ⟺ all σᵢ > 0 ⟺ unique x̂
D4
Rank of A — 6 Ways to Find It
Same number, different computational approaches
▼
#1 Row Reduction Count nonzero pivots in row echelon form — most direct by hand CH1
#2 Col Space Dim rank(A) = dim(col(A)) — dimension of the column space DEF
#3 QR Diagonal Count nonzero diagonal entries of R in QR decomposition CH3
#4 Singular Values Count nonzero σᵢ in SVD — most numerically reliable method SVD
#5 Rank-Nullity Theorem rank(A) = n − dim(null(A)) where n = number of columns THEOREM
#6 Via AᵀA rank(A) = rank(AᵀA) — always true, key for least squares analysis CH2
#7 Eigenvalues (sq.) For square A: rank = number of nonzero eigenvalues only when A is diagonalizable / in special structured settings. In general, safest is pivots or singular values. EIGENVALUES
D5
Norm Conditions that Force Invertibility
The Neumann-series toolbox: “small enough perturbation” cannot destroy invertibility
▼
#1 Neumann Core If ‖C‖ < 1 (for any submultiplicative operator norm), then I-C is invertible and (I-C)^{-1} = I + C + C² + C³ + ... THEOREM
#2 Near Identity If ‖I − A‖ < 1, then A is invertible by writing A = I − (I − A) NORM
#3 Inverse Bound If ‖I − A‖ < 1, then ‖A^{-1}‖ ≤ 1 / (1 − ‖I − A‖) BOUND
#4 Product Case If ‖I − AB‖ < 1 and A,B are square, then AB is invertible, hence A and B are both invertible and full rank CORE
#5 Perturbation Test If A is invertible and ‖E‖ < 1/‖A^{-1}‖, then A+E is invertible because A+E = A(I + A^{-1}E) THEOREM
#6 Spectral Radius Form If ρ(A) < 1, then I−A is invertible since 1 is not an eigenvalue of A EIGENVALUES
#7 DD Route If A is strictly diagonally dominant, then A is invertible (e.g. by Gershgorin: 0 lies in no disc) GERSH.
Master picture: ‖C‖ < 1 ⟹ ρ(C) < 1 ⟹ I−C invertible. The norm test is stronger, but easy to check.

General Splitting Framework, Fixed-Point Form, and Error Equation
The umbrella that contains Jacobi, GS, and SOR
▼
#1 General Splitting Write A = M − N with M invertible. Then Mx_{k+1} = Nx_k + b DEF
#2 Iteration Matrix Equivalent fixed-point form: x_{k+1} = M^{-1}Nx_k + M^{-1}b, so T = M^{-1}N ITERATION
#3 Correctness Any fixed point x* satisfies x^* = M^{-1}Nx^* + M^{-1}b ⟺ (M−N)x^* = b ⟺ Ax^* = b THEOREM
#4 Convergence Test The iteration converges for every starting vector iff ρ(M^{-1}N) < 1 SPECTRAL
#5 Error Equation If e_k = x_k − x^*, then e_{k+1} = Te_k, hence e_k = T^k e_0 ERROR
